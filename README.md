# Bluetooth AutoClick

تطبيق Android للتحكم عن بُعد عبر سماعة البلوتوث.

## الميزات
- زرين طافيين قابلين للتحريك على الشاشة
- ضغطة واحدة على سماعة البلوتوث = تفعيل الزر الأول
- ضغطتين سريعتين = تفعيل الزر الثاني
- نقر تلقائي على مواقع محددة في الشاشة
- توجيه الميكروفون عبر البلوتوث

## المتطلبات
- Android 10 (API 29) أو أحدث

## البناء
يتم بناء ملف APK تلقائياً عند الـ push إلى GitHub.

لتحميل APK:
1. اذهب إلى Actions في GitHub
2. اختر آخر build ناجح
3. حمّل app-debug من Artifacts

## الأذونات المطلوبة
- **Overlay**: لعرض الأزرار الطافية
- **Accessibility**: للنقر التلقائي
- **Bluetooth**: للاتصال بالسماعة
- **Audio**: لتوجيه الميكروفون

## الاستخدام
1. افتح التطبيق وفعّل جميع الأذونات
2. وصّل سماعة البلوتوث
3. اضغط "بدء الخدمة"
4. حرّك الأزرار الطافية للمكان المطلوب
5. اضغط على الزر لتحديد موقع النقر
6. استخدم سماعة البلوتوث للتحكم!
